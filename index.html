CSS Snippet Cheatsheet

Flexbox Row

Use these properties to create a Flexbox row layout. 

                    .row {
                        display: flex;
                        flex-direction: row;
                        flex-wrap: wrap;
                    }

Flexbox Column

Use this to create a Flexbox column layout. 

                    .column {
                        display: flex;
                        flex-direction: column
                    }

CSS Grid Layout

Build a 12-column layout using CSS Grid 

                    .grid {
                        display: grid;
                        width: 100%;
                        grid-template-columns:
                    repeat(12, 1fr);
                    }

Linear Gradients

This will create a background which becomes Linear Gradiant from top to bottom 

                    .linear-gradient-background {
                        background-image: linear-gradient
                        (#da55c3, #9198e5);
                      }

Box Transition Glow

Uses box shadows to glow on hover 

                    .code-card .card-header {
                        border-radius: 30px;
                        transition: all 0.5s ease-in-out;
                      }
                      
                      .code-card:hover, .code-card:hover .card-header {
                        box-shadow: inset 0px 0px 6px grey, 0 0 20px grey;
                      }

Overlay Card with Title

Uses position properties and negative margins to raise elements higher than their natural starting point. 

.card-header {
                    position: relative;
                    margin-top: -10px;
                  }
